<경기 종류> 
  1. 단식 : 1대1 경기로 하고 안쪽 라인을 사용함
  2. 복식 : 2대2 경기로 하고 바깥쪽 라인까지 사용함
  3. 혼합복식 : 남녀 섞어서 2대2 경기로 하고 바깥쪽 라인까지 사용함

<규칙> 
  1. 서브 규칙
	- 서브 위치 : 베이스 라인 뒤쪽에서 대각선으로 있는 서비스 박스에 공을 넣어야 함
	- 두번의 기회 : 첫번째 서브가 실패해도 한 번 더 기회가 주어지며, 두번째 서브도 실패하면 실점(*더블 폴트*)함
	- 서브 순서 : 각 세트의 게임이 끝날 때마다 서브권이 상대에게 넘어감
  2. 기타 규칙
	- 공이 라인에 닿을 때 : 공이 라인에 조금이라도 닿으면 인(in)으로 판정됨
	- 두번 이상 바운드 : 자신의 코트로 공이 넘어온 후 두번 이상 바운드될 때까지 기다리면 상대에게 1점을 줌

폴트(Fault) : 서브를 한 공이 서비스 코트에 들어가지 않거나 서비를 하는 사람이 규칙을 위반한 경우
*더블폴트*(Double fault) : 서브 넣는 선수가 서브를 두번 연속 실수하게 될 때를 말하는데 이럴 경우 2번 실수했기 때문에 서브 넣는 선수는 실점하게 됨

<진행 방식과 득점>
  1. 진행 방식
	- 포인트 → 게임 → 세트 → 매치 순으로 진행
	- 포인트(Point): 테니스 경기에서 가장 기본적인 득점 단위
	- 게임(Game): 한 선수가 4점을 먼저 획득하면 게임을 이김
	- 세트(Set): 한 세트는 총 6게임을 먼저 획득한 선수가 승리
	- 매치(Match): 세트 경기의 총 승패를 결정하는 것으로, 보통 3세트 중 2세트를 먼저 이기거나(3전 2선승), 5세트 중 3세트를 먼저 이기는 방식(5전 3선승)으로 진행
  2. 득점
	- 4번의 포인트 득점으로 한 게임을 이기고, 총 6게임을 먼저 획득해 한 세트를 승리하며, 최종적으로 정해진 세트 수를 먼저 따내는 선수가 경기를 이김
	- 득점 방식은 0, 15, 30, 40 순으로 점수를 세며, 40-40 상황은 '듀스'라고 하여 2점 차이로 먼저 이겨야 게임을 가져갈 수 있음
  3. 득점 방식 (포인트 계산법)
	- 0점 (러브, Love): 아무런 득점도 하지 못한 상태
	- 1점 (피프틴): 15점을 득점한 상태
	- 2점 (서티): 30점을 득점한 상태
	- 3점 (포티): 40점을 득점한 상태
	- 4점 (게임 승리): 4점을 먼저 획득하여 게임을 이김
	- 듀스(Deuce): 양쪽 선수 모두 40-40으로 동점인 상태
	- 애드(Ad): 듀스 상황에서 2점을 먼저 득점하면 게임을 승리합니다. 예를 들어 듀스에서 이기면 '애드인(Ad-in)'이 되고, 지면 '애드아웃(Ad-out)'이 되어 듀스 상황이 계속됨
