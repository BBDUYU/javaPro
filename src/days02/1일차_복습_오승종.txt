1. jdk / jre / jvm
	jdk : 자바로 개발하기 위한 필수요소
	jre : jdk에 포함되어있는 자바 실행프로그램
	jvm : jre에 포함되어있는 자바 가상머신

2. jdk 설치확인 방법 2가지
	1) 설치폴더 확인
	2) cmd -> java -version
	
3. 환경변수 설정
	1) 시스템 속성
	2) 환경변수
	3) 시스템변수 새로만들기
	4) 변수이름 JAVA_HOME, 변수값 디렉터리에서 찾기
	5) PATH에 %JAVA_HOME%\bin 추가 후 위로 올리기
